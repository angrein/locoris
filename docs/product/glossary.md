# Locoris Glossary

## Vault

A complete local workspace with projects, folders, tags, notes, canvases, assets, planner data, settings, and sync metadata.

## Local Vault

A vault stored on the current device.

## Regular Vault

A vault that can sync using plain or encrypted provider behavior depending on configuration.

## Private Vault

A vault that uses client-side encrypted sync and requires a passphrase to unlock synced remote content on each device.

## Remote Vault

A vault record stored through Locoris Cloud, self-hosted sync, or Google Drive.

## Sync Connection

A configured sync method such as Locoris Cloud, self-hosted server, or Google Drive.

## Vault Binding

The relationship between a local vault and a remote vault through one sync connection.

## Locoris Cloud

The managed hosted sync service for Locoris. It is the paid convenience layer in the commercial product direction.

## Self-hosted Sync

The personal server path for users who want to run their own sync endpoint.

## Google Drive Sync

The provider that stores Locoris sync records in Google Drive `appDataFolder`.

## Client-side Encrypted Sync

Sync where vault content is encrypted on the client before being sent to a remote provider.

## Readable ZIP

A portable export intended for human-readable access outside Locoris.

## Exact Backup

A `.locorisbackup` file intended for full Locoris vault restoration.
